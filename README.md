# 高风险客户尽调表生成器使用说明


## 添加到 iPhone 主屏幕

1. 用 iPhone Safari 打开手机访问地址。
2. 点击 Safari 底部分享按钮。
3. 选择“添加到主屏幕”。
4. 以后可直接从桌面图标打开。

## 页面分组

底部导航分为三页：

- 客户：填写机构信息和客户信息。
- 贷款：填写进件时间、亲见日期、贷款信息、核实情况、首付方式和代付关系。
- 完成：手写客户经理签字，查看预览并保存图片。

这样可以减少长页面上下滚动，手机填写时更稳一些。

## 字段排版

页面已把短字段并排处理：

- 所属分中心、所属城市并排。
- 客户姓名、性别并排。
- 客户单位、自雇公司（如有）通过勾选其一选择，再共用一个单位／公司名称输入框；生成图片会保留原表的两个固定栏目，并只把内容写入所选栏目。
- 购车城市、客户电话并排。
- 进件时间、亲见日期并排。
- 获客来源、意向品牌及车型并排。
- 拟贷款成数、贷款金额（万）并排。
- 主要收入来源、年收入情况并排。
- 已进行骗贷提醒、工作和居住地址无异常并排。
- 首付款支付方式、代付关系并排。

生成图片时，表格里的长文本会自动换行并撑开行高，尤其是客户居住地址、单位／公司名称这类内容，不会因为跨行而压到边框。

## 手写客户经理签字

客户经理签字支持手写优先：

1. 在“完成”页的手写签字区写签名。
2. 点击“保存签名”后，签字板会锁定，避免误触。
3. 后续生成其他客户表格时，会自动复用已保存的手写签名。
4. 如需重签，点击“清除签名”，确认后重新书写并保存。

“完成”页不再提供客户经理签字和主管签字输入框，生成图片时客户经理签字只使用手写签字板，主管签字位置留空。

手写签字区域默认收起；点“展开”后即可书写，收起后仍会保留签名状态。

## 保存图片

保存图片有两个入口：

- 右上角下载图标按钮。
- “完成”页底部的“下载图片”按钮，和“清空”并排。

iPhone 的网页和 PWA 不能完全无提示地直接写入相册。点击保存后，系统会优先打开分享/保存面板，你可以在面板里选择保存到照片；如果当前浏览器不支持系统分享，则会退回普通 PNG 下载。

## 清空资料

点击“清空”会清除客户资料并恢复默认字段。

注意：“清空”不会删除已保存的手写签名。如需删除签名，请使用“清除签名”。
