# 高风险客户尽调表生成器使用说明

## 添加到 iPhone 主屏幕

1. 用 iPhone Safari 打开手机访问地址。
2. 点击 Safari 底部的分享按钮。
3. 选择“添加到主屏幕”。
4. 以后可直接从桌面图标打开。

## 填写资料

页面左侧录入客户信息、贷款信息、核实情况、首付方式和日期。右侧会自动生成表格图片预览。

已支持的关键字段包括：

- 客户姓名
- 客户性别
- 身份证号码
- 客户工作单位
- 客户居住地址
- 购车城市
- 客户电话
- 进件时间
- 获客来源
- 意向品牌及车型
- 拟贷款成数及贷款金额
- 年收入情况
- 客户经理签字
- 主管签字
- 亲见日期

## 手写客户经理签字

手写签字优先级高于输入框姓名。

使用方式：

1. 在“手写客户经理签字”区域手写签名。
2. 写完后点击“保存签名”。
3. 保存后签名板会锁定，避免误触修改。
4. 后续生成其他客户表格时，会自动复用已保存的手写签名。
5. 如需重签，点击“清除签名”，确认后重新手写并保存。

如果没有保存手写签名，系统会使用“客户经理签字”输入框中的姓名生成仿手写签名。

## 生成和保存图片

点击“生成图片”可手动刷新右侧预览。

保存图片有两种入口：

- 右上角下载图标
- 下方“下载PNG”按钮

在 iPhone 上，网页/PWA 不能无提示直接写入相册。点击保存后会优先打开系统分享/保存面板，你可以在面板里选择保存到照片；如果浏览器不支持系统分享，则退回普通 PNG 下载。

## 清空资料

点击“清空”会清除客户资料并恢复默认字段。

注意：“清空”不会删除已保存的手写签名。如需删除签名，请使用“清除签名”。
